# 3-Tier Web Application Network Architecture

## Project Overview
Enterprise-grade network design for a secure three-tier web application with VLAN segmentation, inter-VLAN routing, and security policies.

## Lab Objectives
- Design and implement a segmented 3-tier network (Web, App, Data)
- Configure VLANs and inter-VLAN routing
- Implement security policies using ACLs
- Test end-to-end connectivity
